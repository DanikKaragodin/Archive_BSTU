sql
Copy code
INSERT INTO ВашаТаблица (ServiceCode, Firma, OrderDate, OrderCount)

sql
Copy code
DELETE FROM ВашаТаблица
WHERE Firma = 'Фирма 2';
Изменение количества заказов для определенной услуги (например, для услуги с кодом 2):

sql
Copy code
UPDATE ВашаТаблица
SET OrderCount = 10
WHERE ServiceCode = 2;
Выборка всех заказов для определенной фирмы (например, "Фирма 3"):

sql
Copy code

sql
Copy code
SELECT * FROM ВашаТаблица

sql
Copy code
