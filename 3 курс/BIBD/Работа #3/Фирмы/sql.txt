Для таблицы с тремя столбцами "Фирма" (Firma), "Адрес" (Адрес), и "Телефон" (Телефон), вот 6 SQL-запросов для различных операций:

Добавление новой фирмы:


INSERT INTO ВашаТаблица (Фирма, Адрес, Телефон)
VALUES ('Новая Фирма', 'Новый Адрес', 'Новый Телефон');

Удаление фирмы по определенному условию (например, фирму с определенным адресом):


DELETE FROM ВашаТаблица
WHERE Адрес = 'Удаляемый Адрес';

Изменение информации о фирме (например, изменение номера телефона для определенной фирмы):


UPDATE ВашаТаблица
SET Телефон = 'Новый Телефон'
WHERE Фирма = 'Фирма, которую нужно изменить';

Выборка всех фирм с определенным номером телефона:


SELECT * FROM ВашаТаблица
WHERE Телефон = 'Искомый Телефон';

Выборка всех фирм с адресом, содержащим ключевое слово "город":


SELECT * FROM ВашаТаблица
WHERE Адрес LIKE '%город%';

Выборка всех фирм, у которых длина имени (Фирма) больше 20 символов:

SELECT * FROM ВашаТаблица
WHERE LEN(Фирма) > 20;
